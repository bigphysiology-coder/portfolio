# Portfolio — Abdulrasheed Abdulrazak

Full-stack engineer building production web applications with React, Node.js, and PostgreSQL.

Built with React 19, TypeScript, Tailwind CSS v4, Three.js, and GSAP.

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Push to `main` — auto-deploys on Vercel.
